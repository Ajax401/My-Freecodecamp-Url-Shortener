URL Shortener Microservice Courtesy of A Pretorius

Shortening url's one at a time

1. User Story: I can POST a URL to https://shorterweb.herokuapp.com/new/ and I will receive a shortened URL in the JSON response. Example : {"originalurl":"https://www.google.com","shorturl": "hscP7U"}
2. If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"} When I visit the shortened URL, it will redirect me to my original link.
3. Short URL Creation Example Usage: xample: POST https://shorterweb.herokuapp.com/new/https://www.google.com https://shorterweb.herokuapp.com/AWlNU7 Will Redirect to: https://www.freecodecamp.com

This project was build with following libraries node.js, mongoose, url-exist, cors, bodyparser, express, shortid, css ,html.
